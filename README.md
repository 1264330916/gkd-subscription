## 🎯 规则分类

| # | 分类 | 默认开启 | 说明 |
|---|---|---|---|
| 0 | **开屏广告** | ✅ | 自动跳过 App 启动时的开屏广告，无需手动点击「跳过」|
| 1 | **青少年模式** | ❌ | 自动关闭青少年模式弹窗 |
| 2 | **更新提示** | ❌ | 关闭 App 版本更新弹窗 |
| 3 | **评价提示** | ❌ | 关闭应用评分邀请弹窗 |
| 4 | **通知提示** | ❌ | 关闭推送通知授权弹窗 |
| 5 | **权限提示** | ❌ | 关闭权限申请弹窗 |
| 6 | **局部广告** | ❌ | 关闭页面内嵌的局部广告位 |
| 7 | **全屏广告** | ❌ | 关闭全屏覆盖的广告弹窗 |
| 8 | **分段广告** | ❌ | 关闭需要分步骤关闭的广告 |
| 9 | **功能类** | ❌ | 自动执行便捷操作（如自动展开下载线程）|
| 10 | **其他** | ❌ | 未分类的杂项规则 |

> **建议**：至少开启「开屏广告」，其他按需开启。开启太多可能影响正常使用。

## 🏅 全局开屏广告规则

本订阅包含一个强大的 **全局开屏广告规则**，特点：

- **覆盖所有应用** — 即使不在 1,999 个应用列表里的 App，也能自动跳过开屏广告
- **智能匹配** — 识别「跳过」「Skip」「跳過」等关键词
- **防误触** — 排除「搜索」「书签」「视频」等正常按钮
- **10 秒超时** — 只在前 10 秒内生效，不影响正常使用
- **最多点击 2 次** — 避免重复触发

## 📦 安装与使用

### 1. 安装 GKD

- [GitHub Releases](https://github.com/gkd-kit/gkd/releases)（需代理）
- 酷安 / 各大应用商店搜索「GKD」

### 2. 添加订阅

**方法一：手动添加**
1. 打开 GKD App
2. 进入 **订阅管理**
3. 点击 **添加订阅**
4. 粘贴以下链接：
   ```
   https://raw.githubusercontent.com/1264330916/gkd-subscription/main/gkd.json5
   ```
5. 点击 **保存**

**方法二：扫码添加**
> （需要时补上二维码）

### 3. 开启无障碍

1. 手机 **设置 → 无障碍 → 已安装的应用 → GKD**
2. 开启 GKD 的无障碍服务权限
3. 在 GKD 中开启需要的规则开关

## 🔄 更新订阅

订阅会自动检查更新。手动更新：

1. 打开 GKD App
2. **订阅管理** → 点击订阅条目的 **更新** 按钮

> 新增规则会随版本更新自动下发，无需重新添加订阅。

## 🛠️ 本地管理与同步

项目已建立本地 Git 仓库：

```
D:\Github\gkd-subscription\
├── gkd.json5       ← 订阅规则文件（在此编辑）
├── sync_gkd.py     ← 一键同步脚本
└── README.md       ← 本说明文档
```

### 更新流程

1. 在 `D:\Github\gkd-subscription\gkd.json5` 中修改规则
2. 双击桌面 **同步GKD.bat** → 自动上传到 GitHub
3. GKD 手机端手动刷新即可获取最新规则
